# Spondon Blood Donation System

A full-stack blood donation management web application designed to connect donors with recipients, manage donor registrations, blood requests, and facilitate life-saving donations.

## Live Links

- **Frontend (Client)**: [https://spondon-3d428.web.app/](https://spondon-3d428.web.app/)
- **Backend (Server)**: [https://spondon-server.onrender.com/](https://spondon-server.onrender.com/)

## Technologies Used

### Frontend
- React.js (with Vite for build tool)
- Tailwind CSS
- DaisyUI (Tailwind component library)
- JavaScript
- React Icons
- Firebase (for authentication and/or hosting)
- SweetAlert2 (for beautiful alerts)

### Backend
- Node.js with Express.js
- MongoDB (database)
- CORS (Cross-Origin Resource Sharing)
- Dotenv (environment variables management)

## Features
- User registration and authentication
- Donor profile management
- Blood donation requests and searches
- Admin dashboard (if applicable)
- Responsive and modern UI with Tailwind + DaisyUI
- Secure API endpoints for CRUD operations on donations, users, etc.
